CapVisionPro Help Document

Professional camera preview, capture & recording tool with global hotkeys, clipboard auto-sync, and file management.

1. Main Functions

  • Real-time camera preview with adjustable resolution
  • One-click snapshot, auto-save to image/ folder
  • Snapshot automatically copied to system clipboard
  • Video recording with countdown timer and duration display
  • Recording pause & resume function
  • Audio recording support with microphone
  • Quick open image, video, log and ErrorLog folders
  • View latest image or video with one click
  • System tray minimize & toast notifications
  • Complete global keyboard shortcut support

2. Keyboard Shortcuts

Shortcut Function
Alt + C Capture snapshot
Alt + R Start video recording
Alt + O Stop video recording
Alt + Z Pause / Resume recording
Alt + I Open image folder
Alt + V Open video folder
Alt + L Open log folder
Alt + E Open ErrorLog folder
Alt + Shift + S Open latest snapshot
Alt + Shift + V Open latest video
Alt + K Show all shortcuts
Alt + H Open help document
Alt + W / ESC Close subwindow
Alt + P Exit application completely

3. Important Notes

Please make sure your camera is not being used by other software before launching CapVisionPro.
  • Images are saved to image/ directory
  • Videos are saved to video/ directory
  • Normal logs are saved to log/ directory
  • Error logs are saved to ErrorLog/ directory
  • Run the program as Administrator if camera or hotkeys do not work properly
  • Close other camera apps to avoid device occupation issues

4. Troubleshooting

Camera cannot be opened or detected

Check driver installation, camera privacy settings, or close other software that is using the camera.

Hotkeys not responding

Run the program as Administrator, or check for shortcut conflicts with other software.

Recording fails to start

Check disk space, resolution settings, and whether the microphone is available.

Program crashes or errors

Check the ErrorLog/ folder for details and send logs to the developer for support.
